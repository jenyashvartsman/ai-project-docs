# React Docs

## Structure

This documentation set is split into three groups:

- `required/always-on`
- `optional/by-strategy`
- `optional/by-app-type`

Use the required docs as the default baseline for every React app in this repo. Add optional docs only when the app or feature actually uses that strategy.
